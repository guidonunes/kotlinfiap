# Game Tracker (Kotlin FIAP) 🎮

A modern Android application designed to track and manage a personal library of finished video games. This project serves as a practical implementation of modern Android development dynamics and UI design. 

## 🛠 Tech Stack
* **Language:** Kotlin
* **UI Framework:** Jetpack Compose
* **Navigation:** Navigation Compose (handling advanced routing and deep links)
* **Local Database:** SQLite (via Room) 
* **Local Storage:** Shared Preferences (for user settings and lightweight data)

## ✨ Features
* **Finished Games Log:** Keep a persistent record of completed titles (e.g., logging a 100-hour *Baldur's Gate 3* run or a *Black Myth: Wukong* playthrough).
* **Modern UI:** Built entirely with Jetpack Compose for a smooth, reactive user experience.
* **Seamless Navigation:** Utilizes Navigation Compose for fluid transitions between the game list, detailed views, and settings.
* **Data Persistence:** Robust local saving using SQLite to ensure your game library is always available offline.

## 🚀 Getting Started
To run this project locally:

1. Clone this repository: `git clone https://github.com/guidonunes/kotlinfiap.git`
2. Open the project in Android Studio.
3. Sync Gradle to pull down the Navigation Compose and SQLite/Room dependencies.
4. Build and run the app on an emulator or physical device.
